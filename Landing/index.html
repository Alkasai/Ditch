About Ditch

This library can be used by C# clients to make the signing of transactions on the client side and not to transfer private keys to the server.

The essence of the library is to generate a transaction according to the required operations (vote, comment, etc.), sign the transaction and broadcast to the Graphene-based blockchain.

Usage
Choose which blockchain you want to use Steem or/and Golos:
PM> Install-Package Ditch.Steem -Version 3.0.8
PM> Install-Package Ditch.Golos -Version 3.0.8
Add code to open connection:

private readonly OperationManager _operationManager;

public GolosClient(JsonNetConverter jsonConverter) : base(jsonConverter)
{
    //specify how to parse json
    var jss = new JsonSerializerSettings
    {
        DateFormatString = "yyyy'-'MM'-'dd'T'HH':'mm':'ss.fffffffK",
        Culture = CultureInfo.InvariantCulture
    };
    var cm = new WebSocketManager(jss);
    _operationManager = new OperationManager(cm, jss);
}
Add initialization code:
var cUrls = new List { "wss://ws.golos.io" };
var conectedTo = _operationManager.TryConnectTo(cUrls,  CancellationToken.None);

Note: Use correct address https for HttpManager and wss for WebSocketManager.
** TryConnectTo will return url which _operation Manager choose to conect.  If returned null,
 then connection was not established.
Use operation Manager for get or post request, for example:
var config = _operationManager.GetDynamicGlobalProperties(CancellationToken.None);

For more information about supported methods you can visit documentation on Github.
